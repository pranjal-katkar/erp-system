# Objectives & KPIs

## Objectives

1. Build normalized relational database schema.
2. Ensure data integrity through keys and constraints.
3. Provide CRUD operations for all major entities.
4. Implement stored procedures.
5. Create indexing strategy.
6. Complete documentation and ERD.

## KPIs

- Database Normalization: 3NF
- Query Performance: <2 seconds
- Zero orphan records
- 100% ERD completeness
- Minimum 10 stored procedures
- 100% error logging
